# Strategize Phase — 战略制定

## 输入

- `speculo/.speculo/person/<change>/problem-statement.md`（Phase 1 产物）
- `speculo/.speculo/person/<change>/analysis.md`（Phase 2 产物）

- 引语与篇目映射：`references/research/15-quote-bank.md`

## 产物

- `speculo/.speculo/person/<change>/strategy.md`

## 元框架（战略流水线）

**定性（敌我力量本质）→ 定向（主要矛盾）→ 站位（敌我友）→ 时间（阶段/节奏）→ 投放（集中一点）→ 运用（保持主动）。**

Module B 八框架清单、情境选择表与适用条件见 `references/research/12-strategy-frameworks.md`。

## 填写引导

### 1. 确认主框架

对照 Phase 1 确定的 `primary_framework` 和上方的战略快速选择表，确认匹配正确。
若诊断结果改变了问题性质，修正主框架。

### 2. 展开主框架

对选定的主框架详细展开：

- **阶段划分**：当前处于什么阶段？分几步走？
- **关键战役**：哪一仗必须先打赢？为什么？
- **资源配置**：力量往哪投？在哪集中？
- **风险预案**：最坏情况是什么？底线在哪？

### 3. 补充框架

若需要，从 B1-B8 中选取辅助框架（最多 2 个），说明组合方式。

### 4. 写入 strategy.md

```markdown
# 战略方案

## 战略判断
[TODO: 一句话方向判断——"我们的方针是……"]

## 主框架：[框架名]
### 阶段划分
[TODO: 分几个阶段？当前在哪一阶段？]
### 关键战役
[TODO: 必须先打赢哪一仗？]
### 资源配置
[TODO: 力量集中在哪里？]
### 风险与底线

## 辅助框架
[TODO: 如有，说明组合方式]

## 战略箴言

```

## 边界

- 不在此阶段讨论执行细节——那是 Phase 4 的事。
- 战略判断必须给确定性，不能用"可能/也许"回避判断。
- 每一步战略都标注前置条件。

## 完成准则

- 主框架已选定并充分展开
- 阶段划分清晰（至少分两步）
- 有明确的关键战役判断
- `strategy.md` 无残留 `[TODO:]`
