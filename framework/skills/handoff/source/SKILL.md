---
name: handoff
description: 将当前对话压缩为一份交接文档，供另一个 agent 接手。
argument-hint: "下一次会话将用于什么？"
---

编写一份交接文档，总结当前对话内容，以便新的 agent 可以继续工作。保存到用户操作系统的临时目录——而不是当前工作区。

在文档中包含一个「推荐技能」部分，建议下一个 agent 应该调用的技能。

不要复制其他产物（PRD、计划、ADR、issue、commit、diff）中已经包含的内容，改用路径或 URL 引用它们。

删除所有敏感信息，例如 API 密钥、密码或个人身份信息。

如果用户传入了参数，将其视为对下一次会话重点的描述，并据此调整文档内容。
