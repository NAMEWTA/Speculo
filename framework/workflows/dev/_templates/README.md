# Dev Templates

本目录存放 dev 分类下**所有工作流共享**的模板文件。

## 命名约定

- 文件名：`<phase-or-artifact>-template.md`

## 模板文件结构（强制）

**模板不使用 frontmatter。** 顶部用一段引用说明声明归属，然后纯骨架内容：

```markdown
> **服务工作流：** `../<NN-workflow-name>/`
> **产物文件名：** `<最终归档文件名>`

# <标题>

## <章节>
[TODO: 具体填写指引]
```

## 占位符约定（强制）

```markdown
## 项目背景
[TODO: 描述本项目要解决的核心问题、目标用户群体、预期商业价值。1-2 段。]
```

- 必须包含"填什么"的具体提示，禁止无说明的 TODO 占位符
- 填写时整块覆盖，不留残骸
- 可用 `grep -r "\[TODO:" .speculo/` 一键检测未完成模板

## 模板里禁止引导文字

模板纯粹是骨架：**仅顶部归属说明 + 章节标题 + 占位符**。
所有"请按以下要求填写..."的引导文字写在对应 workflow 的 phase `.md` 里（如 `../01-prd/prd-core.md`）。
